# Public Folder - Image Assets

Please add the following images to this folder:

## Required Images:

1. **logo.png** - The circular logo image
   - Should be a square image (recommended: 512x512px or 1024x1024px)
   - Format: PNG with transparent background (preferred) or white background
   - Contains: Book, graduation cap, upward arrow, and gold coins
   - Text: "LEARN FOR EARN"

2. **banner.png** - The horizontal promotional banner
   - Should be a wide image (recommended: 1920x600px or similar aspect ratio)
   - Format: PNG or JPG
   - Contains: Gradient background (sky blue to orange-gold), logo on left, promotional text on right
   - Text: "UNLOCK YOUR POTENTIAL" and "Master New Skills. Achieve Financial Freedom"

## File Naming:
- logo.png (or logo.jpg)
- banner.png (or banner.jpg)

## Usage:
These images will be automatically used by the Logo and Banner components throughout the website.
